# Game Dev Handbook

## Requirements

- Hugo v0.83 or newer
- Hugo extended version, read more [here](https://gohugo.io/news/0.48-relnotes/)

Once you have Hugo installed, you should be able to run the program using 

```bash
hugo serve
```
